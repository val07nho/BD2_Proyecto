Hotel Resort Management System

Dashboard Huespedes Habitaciones Reservas Facturas Preferencias Encuestas

Objetivo

Controlar operaciones de un hotel resort con Oracle y MongoDB.

Oracle

Huespedes, habitaciones, reservas, facturas, pagos, eventos, empleados.

MongoDB

Preferencias, encuestas, historial de estadias y recomendaciones.
